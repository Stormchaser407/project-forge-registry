# recon_housekeeping Export Staging

This folder is for docs-only export staging.

## Allowed

- Markdown notes and approved summaries.

## Not Allowed

- Source code
- Secrets or env values
- Logs, databases, binaries, caches

## Policy

- Keep `_export/docs/` empty until a specific export phase is approved.
- Treat this as a staging lane, not an operational artifact lane.
